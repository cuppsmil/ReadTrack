# ReadTrack - Персональный трекер чтения

Консольное приложение для учёта книг, отслеживания прогресса и анализа читательских привычек. Реализовано на Python в процедурно-функциональном стиле.

## Установка и запуск
1. Создайте виртуальное окружение: `python -m venv venv`
2. Активируйте его: `source venv/bin/activate` (Linux/macOS) или `venv\Scripts\activate` (Windows)
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите приложение: `python src/main.py`

## Функционал
- Каталог книг: добавление, фильтрация, редактирование, удаление
- Прогресс чтения: обновление страниц, процент, прогресс-бар
- Аналитика: еженедельный отчёт, рекомендации по жанрам
- Данные: экспорт (CSV/ZIP), импорт, автоматический бэкап SQLite
